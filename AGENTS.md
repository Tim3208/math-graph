# math-graph 작업 지침

이 파일은 이 저장소의 루트와 하위 파일에 적용되는 작업 지침입니다.

## 프로젝트 방향

- 목적은 수학 문제용 그래프 이미지를 직접 제작하는 웹 도구를 만드는 것이다.
- 초기 단계에서는 과한 구조보다 작고 검증 가능한 변경을 우선한다.
- 앱 코드와 문서 변경은 작고 검증 가능한 단위로 나누어 진행한다.

## 문서 언어

- 기본 문서는 한국어로 작성한다.
- 코드 식별자, 패키지명, 표준 명령어는 원문 영어를 유지한다.
- 공개 사용자에게 필요한 경우에만 영어 병기를 추가한다.

## 구현 원칙

- 현재 앱 스캐폴딩은 Vite, React, TypeScript, Tailwind CSS를 기준으로 한다.
- 그래프 편집 기능은 사용자가 수학 문제 그림을 빠르게 재현하는 흐름을 기준으로 설계한다.
- 그래프 렌더링은 SVG를 우선한다. Canvas 도입은 대량 객체나 자유 드로잉 요구가 명확해진 뒤 검토한다.
- 기능 추가 시 기존 패턴을 우선하고, 새 의존성은 필요성이 명확할 때만 추가한다.
- 생성물과 런타임 상태는 저장소에 포함하지 않는다.

## Git 및 검증

- 기본 브랜치는 `main`을 사용한다.
- 커밋 메시지는 상위 지침의 Lore Commit Protocol을 따른다.
- 변경 후에는 가능한 가장 작은 검증 명령부터 실행한다.
- `.omx/`는 로컬 런타임 상태이므로 Git에 포함하지 않는다.

## 제외 대상

- `.omx/`
- 의존성 디렉터리
- 빌드 산출물
- 로그 파일
- 로컬 환경 변수 파일
