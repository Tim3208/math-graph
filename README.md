# math-graph

수학 문제에 등장하는 그래프를 직접 그리고, 필요한 점과 선분, 보조선, 축 표시를 빠르게 추가하기 위한 그래프 제작 도구 프로젝트입니다.

## 목적

문제 해설이나 수업 자료를 만들 때 원하는 그래프 이미지를 매번 찾고 캡처하는 과정을 줄이는 것이 목표입니다. 사용자는 좌표평면 위에 함수 그래프, 꺾은선, 점, 선분, 점선 보조선, 라벨 등을 직접 배치해 문제 상황에 맞는 그림을 만들 수 있어야 합니다.

## 현재 단계

이 저장소는 Vite, React, TypeScript 기반의 웹앱 뼈대를 갖춘 초기 개발 단계입니다. 현재는 단일 변수 수식 `y=f(x)`를 SVG 좌표평면에 렌더링하는 첫 MVP를 구현했습니다.

현재 포함된 항목:

- 프로젝트 설명 문서
- 저장소 작업 규칙
- MIT 라이선스
- Git 추적 제외 설정
- Vite + React + TypeScript 앱 설정
- Tailwind CSS 기반 스타일링 설정
- SVG 그래프 편집 화면의 초기 레이아웃
- `y=f(x)` 수식 입력, 축 범위/눈금 입력, 오류 표시
- 수식 샘플링과 SVG path 렌더링 로직 테스트

## 로컬 개발

의존성을 설치한 뒤 개발 서버를 실행합니다.

```bash
npm install
npm run dev
```

프로덕션 빌드는 다음 명령으로 확인합니다.

```bash
npm run build
```

순수 계산 로직 테스트는 다음 명령으로 실행합니다.

```bash
npm run test -- --run
```

## 예정 기능

- 좌표축과 눈금이 있는 그래프 캔버스
- 여러 수식 그래프 관리
- 점, 선분, 꺾은선, 점선, 보조선 표시
- 좌표와 수식 라벨 추가
- 이미지로 내보내기
- 자주 쓰는 그래프 템플릿 저장
- 수학 문제용 깔끔한 흑백 스타일과 발표 자료용 컬러 스타일

## 개발 방향

프론트엔드는 React, TypeScript, Vite 기반의 단일 페이지 앱으로 시작합니다. 그래프 편집 영역은 축, 점, 선, 라벨을 개별 객체로 다루기 쉬운 SVG 우선 구조로 설계합니다. 수식 파싱은 `mathjs`의 parse/compile 기반 평가를 사용합니다.
