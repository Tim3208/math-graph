# math-graph

수학 문제에 등장하는 그래프를 직접 그리고, 필요한 점과 선분, 보조선, 축 표시를 빠르게 추가하기 위한 그래프 제작 도구 프로젝트입니다.

## 목적

문제 해설이나 수업 자료를 만들 때 원하는 그래프 이미지를 매번 찾고 캡처하는 과정을 줄이는 것이 목표입니다. 사용자는 좌표평면 위에 함수 그래프, 꺾은선, 점, 선분, 점선 보조선, 라벨 등을 직접 배치해 문제 상황에 맞는 그림을 만들 수 있어야 합니다.

## 현재 단계

이 저장소는 초기 준비 단계입니다. 아직 웹앱 코드, API, 패키지 설정, 타입 정의는 포함하지 않습니다.

현재 포함된 항목:

- 프로젝트 설명 문서
- 저장소 작업 규칙
- MIT 라이선스
- Git 추적 제외 설정

## 예정 기능

- 좌표축과 눈금이 있는 그래프 캔버스
- 점, 선분, 꺾은선, 점선, 보조선 표시
- 좌표와 수식 라벨 추가
- 이미지로 내보내기
- 자주 쓰는 그래프 템플릿 저장
- 수학 문제용 깔끔한 흑백 스타일과 발표 자료용 컬러 스타일

## 개발 방향

다음 단계에서 프론트엔드 스캐폴딩을 별도 계획으로 진행합니다. 초기 후보는 React, TypeScript, Vite 기반의 단일 페이지 앱이며, 그래프 편집 영역은 SVG 또는 Canvas 기반 구현을 비교한 뒤 결정합니다.
